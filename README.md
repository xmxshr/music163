# 网易云音乐（移动端）

使用 jQuery 实现移动端网易云音乐，包含首页、推荐歌单、播放器三个页面  
[预览链接](http://xumx.site/)  
[![1514712938.png](https://i.loli.net/2017/12/31/5a48af73a614f.png)](https://i.loli.net/2017/12/31/5a48af73a614f.png)
  
### 实现的功能
- 音乐播放、暂停
- 搜索单曲、歌手、专辑
- 热门搜索
- 搜索历史记录
- 排行榜
- 推荐歌单
- 歌词(&歌词翻译)  

### 前端
 - jQuery
 - 动态rem

### 数据库
 - leanCloud
